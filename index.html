KhanomTan TTS model v1.0

a open-source Thai text-to-speech model.

KhnomTan TTS (ขนมตาล) is the first open-source Thai text-to-speech model that supports multilingual speakers. It supports Thai, English, and others.

KhnomTan TTS is a YourTTS model that trained with supports Thai. We add the Thai speech corpus from TSync 1* and TSync 2* to mbarnig/lb-de-fr-en-pt-12800-TTS-CORPUS that train the model with YourTTS model.
